# **App Name**: AquaTrack

## Core Features:

- Assessments Display: Assessments Tab: Displays swim class assessment entries with fields for Teacher Name, Class Time, Assessment Date, and Assessor Initials. Allows filtering by day of the week using filter chips and groups classes by Teacher.
- Assessment Completion: Assessments Completion: Tick button or checkmark next to each class to mark the assessment as complete, updating the assessment date or toggling a completed state. Automatically mark assessments older than 4 weeks as overdue with visual highlighting.
- Missed Assessments: Missed Assessments Tab: Displays missed assessment entries including Student Name, Teacher Name, Class Time, Assessment Date, and Assessor Initials.  The ASO can use text input to search, sort and filter this list.
- Waitlist Management: Waitlist Tab: Displays waitlist entries including Student Name, Class Level, Preferred Time, Date Added, ASO Initials, and Notes.  Enables sorting by most recent entry or preferred time.
- Access Control: Access Control: Initially keeps the app private, with the option for the owner to adjust access later.

## Style Guidelines:

- Primary color: Deep turquoise (#45B6FE), evokes clarity and safety, fitting for aquatics.
- Background color: Light cyan (#E0F7FA), offers a clean, calming backdrop that won't distract users.
- Accent color: Muted indigo (#5D8AA6), highlights key interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, known for its clean, modern, objective, and neutral aesthetic.
- Tabular layouts with clear headings and labels for easy data entry and viewing.
- Ensure list density is optimized for quick scanning of information on mobile and desktop.
